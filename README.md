# DeepPPL
Deep Probabilistic Programming Language


## Installation
```
git clone https://github.com/deepppl/deepppl.git
cd deepppl/deepppl
make
pip install -r requirements.txt
pip install ..
```

To create a symbolic link to the code use: `pip install -e ..` instead
